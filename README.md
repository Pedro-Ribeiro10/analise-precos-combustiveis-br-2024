⛽ **Análise Semestral de Preços de Combustíveis no Brasil (2024)**

📝 **Descrição**
Este projeto apresenta uma análise detalhada dos preços dos combustíveis automotivos no Brasil, utilizando dados oficiais e públicos referentes ao ano de 2024. O objetivo foi transformar um grande volume de dados governamentais em um dashboard intuitivo para monitoramento de mercado e comparação de preços por estado e bandeira.

📊 Funcionalidades do Dashboard
 • **Visão de Preços:** Monitoramento do preço médio da gasolina (comum e aditivada), Diesel, Etanol e GNV.

 • **Análise por Bandeira:** Comparação de performance de mercado entre bandeiras (Branca, Petrobras, TotalEnergies, Royal Fic, Stang, etc.).

 • **Extremos de Mercado:** Identificação automática dos estados com o preço máximo e mínimo por produto.

 • **Série Temporal:** Média geral de preços com granularidade diária.

🛠️ **Tecnologias**
 • **Power BI:** ETL (Power Query), Modelagem de Dados e DAX para cálculos de médias móveis e extremos.

 • **Fonte de Dados:** Dados abertos do Governo Federal (Série histórica de preços de combustíveis).
